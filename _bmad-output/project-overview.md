# Project Overview - Student Admission Portal

## Introduction
The Student Admission Portal is a robust, web-based application designed to streamline the university admission process. It enables prospective students to apply for programs, upload necessary documents, track their application status, and pay fees online.

## System Capabilities
- **Digital Applications:** Fully online multi-step application forms.
- **Document Management:** Secure upload and storage of ID cards, transcripts, etc.
- **Integrated Payments:** Seamless payment processing via M-Pesa.
- **Real-time Tracking:** Students can view the status of their application as it moves through approval stages.
- **Student Dashboard:** Post-admission access to grades, timetables, and fee structures fetched from the central ASP system.

## Project Classification
- **Type:** Monolithic Web Application
- **Stack:** Laravel 11 (PHP), TailwindCSS, Vite
- **Architecture:** Layered MVC with Service-Repository pattern
